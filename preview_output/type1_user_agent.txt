[user_agent] Sample #1  (topic=工作職場)
======================================================================
System Prompt:
  你是一個人工智慧助理，精通心理調適與職場技能。請指導使用者如何迅速緩解焦慮，並提供增強自信的策略。

  User：我下午要去作重要報告，但現在覺得很緊張不安，有沒有什麼快速放鬆心情和增強信心的辦法啊？
  Agent：深呼吸三到五分鐘有助於放鬆，想像成功情景也能提昇自信心。
  User：哦這樣，我馬上試試深呼吸。
  Agent：很好，做幾次深呼吸後，可以把報告內容簡述一遍，熟悉流程也有助於增加信心。
  User：我知道了，然後如果有空的話，再去走動一下嗎？感覺運動也可以讓身心放鬆一些。
  Agent：當然可以，走動一下或是簡單伸展都能幫助放鬆，提高精神集中度。
  User：好的，那我在準備報告之餘，也會抽空起來活動活動筋骨。
  Agent：這樣很好，保持身體舒暢，心情會更輕松，報告時表現也會更加出色。

[user_agent] Sample #2  (topic=教育)
======================================================================
System Prompt:
  你是一個人工智慧助理，專精於教育與學習規劃。請根據五年級數學需求，為家長推薦合適的補習班，並說明選取標準。

  User：我家五年級小孩數學不太好，想找個補習班補救一下，怎樣選比較好啊。
  Agent：選補習班要看教師資歷和教學評價，還有是否採小班制教導。
  User：嗯，這些都是重要考量點。
  Agent：另外也可以考慮交通方便性和上課時間是否符合家庭安排。
  User：對，這部分我也要注意一下。附近有哪些不錯的補習班你知道嗎？
  Agent：我不確定，但你可以在網路搜尋或參考當地家長社群的推薦。
  User：好，我就上網查檢視周圍有些啥補習班。
  Agent：查時留意學生反饋和補習班提供的免費體驗課程，這樣更瞭解實際情況。
  User：嗯嗯，聽起來像是很好的建議，我可以去試著申請免費體驗課程看看。
  Agent：是的，透過體驗課程可以直接感受教學風格和環境，很實用呢。
  User：好的，那開始行動吧，先在網路上找出幾家口碑不錯的補習班。
  Agent：很好，記得比對各機構的強弱項，找到最適合你們的那一間。
  User：我知道了，我會仔細比較他們的特色的。
  Agent：對，每個補習班都有自己的特長，找到符合孩子需求的就是最好的。
  User：確實如此，那就依照之前提到的條件去找找看了。
  Agent：對，按那些條件挑選，相信你能找到理想的補習班。

[user_agent] Sample #4  (topic=假期旅遊)
======================================================================
System Prompt:
  你是一個人工智慧助理，熟悉臺灣旅遊景點。請為使用者推薦幾處風景優美且適合露營的海岸線，供其環島腳踏車之旅參考。

  User：想找一些風景漂亮又適合搭帳篷的海岸線，可以推薦嗎？最好是臺東或者花蓮那邊的。
  Agent：三仙臺和綠島都是不錯的選擇，風景美又適合露營。
  User：哦這樣，那這些地方附近有沒有比較方便租借腳踏車的地方啊？
  Agent：可以在成功鎮或是瑞源漁港附近的店家租借腳踏車，很方便。
  User：好，那要去三仙臺或者綠島的話，需要提前訂票還是可以直接過去呢？
  Agent：綠島需預定船票，三仙臺則可直接前往，但假日人多最好早去。
  User：明白了，再去之前應該要做哪些安全準備呢？像是急救包還是防曬用品這類的。
  Agent：帶上防曬霜、帽子、太陽眼鏡、防蚊液、急救包、水和零食、帳棚、燈具，多穿輕便衣褲。
  User：好的，聽起來準備清單滿長的耶，不過為了安全萬萬不可少，謝了。
  Agent：確實很重要，做好準備才能玩得開心又安全。
